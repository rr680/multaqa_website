Multaqa API Documentation

This server hosts the Multaqa API. The frontend application needs to be deployed separately or accessed through its designated URL.

Key Endpoints

GET /events

Fetch all events

GET /health

API health check endpoint

Next Steps

To deploy the frontend:

  • Copy the build files to the public directory of this server, or
  • Deploy the frontend to a separate hosting service
